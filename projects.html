Projects by Muhi Eddin Tahhan

Vader

  • Open-source cybersecurity project.
  • Used Python libraries to implement a bunch of cybersecurity tools based on the knowledge obtained from pursuing the cybersecurity field.
  • Tools implemented: Hash Cracker, Port Scanner, Subdomain Enumerator, SSH Brute Force, and Directory Enumerator.
  • Built a command line interface to help users interact with the program.
  • Check out the documentation for more details:
    • https://github.com/MuhiEddinTahhan/vader

Local Shop App

  • Made the application using Kotlin for the back end, XML for the front end, and Fire Store for the database.
  • Implemented using Object Oriented Programming.
  • Implemented an authentication system using email and password. The information is stored in the Fire Store database.
  • Added features like a cart for the bought products, uploading products to other users, creating, and updating profiles, etc.
  • Check out the documentation for more details:
    • https://github.com/MuhiEddinTahhan/localshop

Flying Spacecraft

  • An Action-Adventure game where you drive a spacecraft through the map and try to escape the enemies.
  • Used many of Unity’s utilities to build the logic, the map, and other characters in the game like Timeline, Terrain, and Unity’s particle system.
  • Wrote scripts in C# to describe the movements and the actions of every character in the game.
  • Utilized AI to create the music for the game in addition to the helper character.
  • Check out the documentation for more details:
    • https://github.com/MuhiEddinTahhan/Flying_Spacecraft

Virus Simulator

  • A project that is designed to showcase my knowledge in Java and software design patterns.
  • Used many of Unity’s utilities to build the logic, the map, and other characters in the game like Timeline, Terrain, and Unity’s particle system.
  • Built The application by designing interfaces and abstract classes to build the concrete classes.
  • Implemented Singelton, Observer, Factory, Template, and Strategy design patterns throughout the program.
  • Utilized data structure like Array Lists and hash maps to design some of the features in the simulation.
  • Check out the documentation for more details:
    • https://github.com/MuhiEddinTahhan/Virus-Simulation
